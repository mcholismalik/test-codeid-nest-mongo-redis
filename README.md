# Constanta API

CRUD API with NestJS + MongoDB + Redis

## Installation

```bash
$ npm install
```

## Run

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Guide

- Open swagger document

```bash
http://localhost:3001
```
